// ИМПОРТ СПИСКА УЧАСТНИКОВ (TG DESKTOP)

Telegram официально не отдаёт getChatMembers через Bot API. Чтобы увидеть всех ~32 фактических участников группы, экспортируйте список вручную:

1. Откройте Telegram Desktop → нужная группа → ... → View Group Info → N Members.
2. Прокрутите список до конца — Telegram подгрузит всех.
3. Скопируйте список (правой кнопкой → Copy / в Telegram Desktop) или используйте инструмент «Export chat history» (Settings → Advanced → Export Telegram data → конкретная группа, только Members JSON).
4. Вставьте сюда в формате JSON-массива.

Поддерживаемые форматы:
[{"telegram_id":118296372,"username":"tyoma_fyodorov","first_name":"Artyom"}]
Поля: telegram_id обязательное; username, first_name, last_name, status, is_admin опциональные.
